# Raw Sources — SymGPT: Smart Contract Auditing via LLMs + Symbolic Execution

## Auditing Smart Contracts via Combining Symbolic Execution with Large Language Models
- **URL**: https://arxiv.org/html/2502.07644v2
- **Type**: primary


## Candidate Summary
SymGPT combines natural language understanding of LLMs with formal guarantees of symbolic execution to verify ERC rule compliance. Identified 5,783 ERC violations in 4,000 real contracts, including 1,375 with clear attack paths for stealing assets.

## Why It Matters
Novel approach to automated smart contract auditing that outperforms existing tools and expert services. Addresses critical need for scalable ERC compliance verification given complexity of 132 ERC rules across common standards.
