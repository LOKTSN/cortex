## Candidate Summary
OpenFang is a full Agent Operating System built in Rust (137K LOC, 14 crates) that compiles to a single ~32MB binary. Introduces 'Hands' - pre-built autonomous capability packages (Researcher, Lead, Collector, Clip, Predictor, Twitter, Browser) that run 24/7 without user prompts.

## Why It Matters
Represents a new paradigm in agent frameworks - moving beyond chatbot-style interaction to continuously running autonomous agents. Includes 16 discrete security layers, 40 channel adapters, and WASM sandboxing. Significantly faster cold start (180ms) vs competitors.
