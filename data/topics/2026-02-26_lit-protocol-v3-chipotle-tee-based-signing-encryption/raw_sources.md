## Candidate Summary
Lit Protocol launched v3 'Chipotle', a ground-up rebuild featuring TEE-based execution instead of threshold cryptography. The upgrade delivers programmable signing/encryption via standard REST API, eliminating SDK dependencies. Designed for AI agents and HTTP-native workflows with on-chain key management.

## Why It Matters
Major architectural shift from multi-node threshold signing to single TEE execution - dramatically reduces latency and cost while enabling AI agents to use programmable cryptographic keys via simple HTTP calls. This could accelerate agentic DeFi workflows.
