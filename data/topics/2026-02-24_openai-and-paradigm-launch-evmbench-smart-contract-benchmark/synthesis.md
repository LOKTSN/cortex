**TL;DR:** OpenAI and Paradigm have launched EVMbench, the first standardized benchmark for evaluating AI agents in smart contract security auditing. The benchmark tests AI capabilities across detecting, patching, and exploiting vulnerabilities using 120 curated flaws from 40 real-world audits. GPT-5.3-Codex achieved 72.2% in exploit mode versus 31.9% for GPT-5, highlighting significant strides in AI-driven vulnerability detection as smart contracts secure over $100B in assets.

**Key Insights:**

- **Benchmark Design:** EVMbench focuses on the Ethereum Virtual Machine (EVM) ecosystem, covering common vulnerability classes like reentrancy, integer overflow, and access control flaws. It evaluates AI agents in three modes: detection, patching, and exploitation—providing a comprehensive assessment of offensive and defensive capabilities.

- **Performance Gaps:** The results reveal a substantial performance gap between model versions. GPT-5.3-Codex's 72.2% exploit success rate compared to GPT-5's 31.9% demonstrates that newer reasoning-heavy architectures are significantly better at understanding contract logic and crafting exploit payloads—a critical advancement for automated security auditing.

- **Industry Implications:** With $100B+ locked in smart contracts, the lack of standardized AI security benchmarks has been a major gap. EVMbench enables systematic evaluation of AI tools, paving the way for more reliable AI-assisted audits. OpenAI's additional $10M API credit commitment for cyber defence signals broader support for securing critical infrastructure.

**Why this matters:** EVMbench provides the first rigorous framework to measure AI's real-world utility in securing smart contracts—a vital step as the ecosystem grows and automated vulnerability detection becomes essential for protecting billions in digital assets.
