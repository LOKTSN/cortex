## Candidate Summary
Microsoft Agent Framework has reached RC status for both .NET and Python, merging the previously separate AutoGen and Semantic Kernel into a unified open-source SDK. The framework provides multi-agent workflows, function tools, and supports A2A, AG-UI, and MCP protocols.

## Why It Matters
This marks the end of fragmentation in Microsoft's agentic AI tooling. Developers previously using either AutoGen or Semantic Kernel now have a consolidated path forward. GA expected Q1 2026.
