## TL;DR

0x has launched its Cross-Chain API to private beta, enabling token swaps across 15+ blockchains through a single integration. The API supports EVM networks and Solana, aggregates liquidity via multiple bridge providers, and settles common pairs in under a minute—targeting AI agent execution workflows and autonomous trading systems.

---

**1. Unified Cross-Chain Execution for Autonomous Agents**

The API consolidates fragmented cross-chain infrastructure into a single endpoint. Developers can now execute swaps across EVMs and Solana without managing multiple bridge integrations, aggregators, or chain-specific logic. Liquidity aggregation across bridges ensures competitive pricing and reduces slippage for agent-driven trades.

**2. Native Support for Emerging Agent Payment Standards**

The integration aligns with emerging payment protocols: Coinbase's x402 (request-for-payment), Google's AP2, and OpenAI's ACP (Agent Communication Protocol). This positions 0x as infrastructure for AI agents that need programmatic, self-custodial payment flows—critical for autonomous DeFi strategies.

**3. Speed and Reliability for Time-Sensitive Strategies**

Sub-minute settlement for common pairs addresses a key bottleneck for agents executing cross-chain strategies. Speed matters for arbitrage, yield optimization, and liquidity provision—where delays mean missed opportunities or liquidations.

---

**Why this matters:** This release bridges DeFi infrastructure and agentic AI, providing the executable liquidity layer autonomous systems need to operate across fragmented chains—accelerating the shift toward self-driving financial agents.
