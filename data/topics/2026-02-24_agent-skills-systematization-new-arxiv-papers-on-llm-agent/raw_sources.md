## Candidate Summary
Multiple new arXiv papers examine agentic skills in LLMs: 'Agent Skills for LLMs' (2602.12430) explores modular skill-equipped agents, 'SoK: Agentic Skills' (2602.20867) systematizes beyond tool use, and 'Tool-R0' (2602.12430) presents self-evolving agents for zero-shot tool learning.

## Why It Matters
Indicates emerging academic focus on formalizing agent architectures and skill acquisition. Tool-R0's self-evolving capability addresses the limitation of requiring prior training data - significant for production agent deployments.
